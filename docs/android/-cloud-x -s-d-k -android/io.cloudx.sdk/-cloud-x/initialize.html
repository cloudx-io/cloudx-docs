CloudX SDK Android/io.cloudx.sdk/CloudX/initialize

initialize

@JvmStatic
@JvmOverloads
fun initialize(activity: Activity, initializationParams: CloudX.InitializationParams, listener: CloudXInitializationListener? = null)

Initializes CloudX SDK; essential first step before loading and displaying any ads.

If the SDK is already initialized, the provided listener will receive CloudXInitializationStatus.initialized - true.

After a successful initialization, you can:

  • createBanner

  • createMREC

  • createNativeAdSmall

  • createNativeAdMedium

  • createInterstitial

  • createRewardedInterstitial

Parameters

initializationParams

initialization credentials and misc parameters

listener

an optional listener to receive initialization status updates

See also

CloudX.isInitialized
